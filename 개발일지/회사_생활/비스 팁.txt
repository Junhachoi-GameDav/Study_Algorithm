비주얼 스튜디오 에서 지원해주는 것들

밑에 창으로 조사식, 로컬, 메모리 등으로 변수의 값과 위치 주소 등을 볼수있다.

빌드 이후 런(f5)를 하기전 코드 왼쪽을 클릭하면 디버깅 브레이크 표시가 뜬다. (빨간 점)
이후에
 f12 = 함수 정의로 이동함 or ctrl + 마우스 클릭
 f10 = 다음 코드 줄로 이동 (동작중에)
 f11 = 코드 참조로 이동함  (동작중에)

ctrl + f 를 누르면 코드 찾기 ctrl + shfit + f를 누르면 전체 코드 찾기

함수명 쉽게 변경하기, 컨에프로 바꾸면 딴것도 바뀐다. Ctrl + R + R (rename)  

ctrl + k + c 주석 ctrl + k + u 주석 해제

뒤로 탐색, 앞으로 탐색 = Ctrl + (-), Ctrl + Shift + (-) 

자동 정렬하기 = Ctrl + K + F 

Tab (또는 두번) 자동완성

***
디버깅 브레이크 지점에 오른쪽 마우스 클릭하면 조건식을 넣어서 브레이킹 할수 있음
