1. 스택과 메모리 와 비슷하지만 더 세세히 들어가면...

전역변수는 어디서든 사용가능하지만 큰 규모일수록 사용하면 안되서 지역변수를 자주 사용하는데
지역변수는 선언후 그것을 수정할때 사실상 복사를 하는것이다.~~~~!!!!!!
즉 스택에 변수들이 올라갈때 지역변수가 선언되고 함수로 수정하는것은 복사된 변수를 쓰는것이라 원본은 바뀌지 않는다.

ex

void add_hp(int hp, int value)
{
    hp += value;
}

int main()
{
    int hp = 100;              // 지역변수
    add_hp(hp, 20);
    cout << hp;
}

결과 100

120이 나와야 할것 같지만 지역변수는 그대로 100이고 add_hp함수에 복사가되어 120으로 만들어도 결국 cout에서
원본 hp 100을 불러왔기 때문에 100이 출력 되는 것이다.
즉 항상 알고있듯이 지역변수는 그 함수안에서만 사용가능하고 다른 함수 안에서는 수정이 불가능하다는것을 숙지 해야한다.
