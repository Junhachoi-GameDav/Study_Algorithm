요청 -> 처리 -> 응답

제일 많이 쓰는 단축키 모음

1. Alt + Enter
 - 수정 제안 목록이 표시.

2. F2
 - 다음 오류, 경고 또는 제안으로 점프.

3. Alt+1
 - 도구 창을 염.

4. Ctrl+E
 - 최근 실행했던 파일을 확인.

5. Ctrl+B
 - 심볼의 선언으로 이동

6. Ctrl+Alt+B
 - 구현으로 이동

7. Alt+F7
 - 어떤 항목이 사용된 모든 곳이 표시

8. Ctrl Ctrl
 - Run Anything 창이 열림(어떤 항목이든 실행하는 창)

9. Ctrl + W    or    Shift + Ctrl + W
 - 선택 영역 확대

10. Ctrl + /    or    Shift+Ctrl+/
 - 해당 코드 줄이 줄 주석로 처리
 
11. Shift+Ctrl+Enter
 - 코드 자동완성

12. Ctrl+Alt+L    or    Shift+Ctrl+Alt+L
 - 해당 프로젝트의 표준에 맞도록 현재 파일의 서식을 지정( 줄 틀린거, 괄호 빼먹은거 등등 고쳐줌)

13. Shift+Ctrl+A
 -  Find Action 창을 띄움(모든 액션을 검색)

14. Shift Shift
 - Search Everywhere창을 띄움(전체 검색)
